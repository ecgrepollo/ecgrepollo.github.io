<h1>Brief Explanation of the application</h1><br><br>
The application has [Login], [Inventory], [Point of Sales], [Transaction], [View Orders], [Reports], [Account], [Critical Items]<br><br>
<h2>[LOGIN]</h2><br>
<br><br>
<h2>[INVENTORY]</h2><br>
The user can add / delete / adjust items in the inventory. Inventory form also allows the user to add product name and also the supplier's info like name and TIN.<br><br>
<h2>[VIEW ORDERS]</h2><br>
After adding items in the inventory. The information is stored in the database and can later be viewed in the View Orders form.<br><br>
<h2>[POINT OF SALES]</h2><br>
The point of sales form displays the current inventory available to sell. It also has its "cart" and computes the total amout of what's inside the cart. The user can also make the items discounted.<br>
Point of Sales can also print receipts.<br><br>
<h2>[TRANSACTIONS]</h2><br>
All transactions that is done at the Point of Sales form is stored in the database.<br>
The user can make a refund transaction inside this form<br><br>
<h2>[REPORTS]</h2><br>
This Form displays three reports for now.<br>
[1] Ending Report - displays total sales for the day along with remaining inventory.<br>
[2] Ending Report (Multiple Dates) - displays total sales from a specific period of time.<br>
[3] Expiration of Items - Displays all available items in the inventory with their expiration dates.<br><br>
<h2>[ACCOUNTS]</h2><br>
Only admins can use this form. This form allows the admin to create and edit accounts' access in the application.<br><br>
<h2>[CRITICAL ITEMS]</h2><br>
This form prompts the user as soon as the user logs in if there's any item that needs replenishment or there are some items that are near its expiration dates.<br><br>
